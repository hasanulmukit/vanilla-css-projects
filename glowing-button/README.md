# Enhanced Glowing Buttons

A visually striking project featuring two styles of glowing buttons. These buttons are designed to captivate users with their smooth animations and glowing effects.

## Features

- **Glowing Button**: Dynamic gradient glow animation that radiates an elegant aura.
- **Neon Button**: Retro-inspired neon style with glowing borders and responsive hover effects.
- **Modern Background**: Radial gradient background for a polished aesthetic.
- **Responsive Design**: Layout adapts to different screen sizes.
